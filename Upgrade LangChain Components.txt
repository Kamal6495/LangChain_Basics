Step 1: Upgrade LangChain core packages
---------------------------------------
pip install -U langchain langchain-community

Step 2: Install LangChain CLI (for upgrade tool)
------------------------------------------------
pip install langchain-cli

# Verify installation
langchain --help

# If CLI still not recognized in PowerShell, use:
python -m langchain --help


Step 3: Upgrade imports in your code
------------------------------------
# Upgrade a single Python file
langchain upgrade D:\Practice\Thesis\3_LangChainBasics\8.Runnables\2b_RetriverQAChain_docstxt_reader.py

# (Optional) Upgrade all files in a folder
langchain upgrade D:\Practice\Thesis\3_LangChainBasics\8.Runnables\

# (Optional) Preview changes without editing
langchain upgrade --dry-run D:\Practice\Thesis\3_LangChainBasics\8.Runnables\2b_RetriverQAChain_docstxt_reader.py


Step 4: Update Python dependencies (via proxy)
----------------------------------------------
# Install all packages from requirements.txt
pip install -r requirements.txt --proxy http://edcguest:edcguest@172.31.102.14:3128

# Upgrade all packages to latest compatible versions
pip install --upgrade -r requirements.txt --proxy http://edcguest:edcguest@172.31.102.14:3128

# Save updated versions back into requirements.txt
pip freeze > requirements.txt


Step 5: Run your script
-----------------------
python D:\Practice\Thesis\3_LangChainBasics\8.Runnables\2b_RetriverQAChain_docstxt_reader.py
